Instructions

  1. Grab a colourful object
  2. Hit record!
  3. Calibrate an object by holding in the green square (the selected pixels should go black). When you are happy - click the 'calibrate' checkbox.
  4. To submit a line, start in the bottom left box, do some art, then end in the bottom right.
  5. Draw!
